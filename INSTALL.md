# Instructions:
User is assumed should have ocaml and opam installed; however, if the user does
not, then follow installation instructions according to the [OCaml textbook](https://cs3110.github.io/textbook/chapters/preface/install.html).

To play the game, run `make play` in the terminal.

# Game Rules
User can input `play [card name]` to play a card from their hand.\
User can input `end` to quit the game.

# Demo Rules
Currently, you are on the slime floor which is the weakest enemy in the game.
The only cards in your deck are *OMEGA attack* and *baby attack*.
